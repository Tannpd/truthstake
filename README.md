# TruthStake — Decentralized Fact-Checking Market

TruthStake is a decentralized betting and insurance platform against fake news, where GenLayer's network of AI Validators automatically cross-checks facts across reputable news sites. 

### Why TruthStake DIES without GenLayer
> **Traditional blockchains (like Ethereum or Solana) cannot natively scrape the live web or process natural language prompts, meaning TruthStake would die without GenLayer's unique ability to connect AI nodes directly to live web data and achieve consensus on semantic outcomes rather than strict string matching.**

---

## Features
- **Multi-source Fact-checking**: AI validators scrape 3-5 news URLs from whitelisted reputable domains (`reuters.com`, `apnews.com`, `bloomberg.com`, `nytimes.com`, `bbc.com`, `bbc.co.uk`) using `web.render`.
- **Semantic Consensus**: A custom `validator_fn` compares the logical verdicts of the validator nodes (TRUE vs. FALSE) rather than strict JSON string comparison, ensuring consensus is reached even with slight wording or formatting differences.
- **Robust Error Handling**: Gracefully handles paywalled articles, unreachable URLs, and parsing failures by placing markets into refund states.
- **Pull-Style Payouts**: Proportional reward distribution with a secure pull pattern, protecting the contract against re-entrancy and gas-limit exploits.

## 1. How to Deploy the Intelligent Contract

To deploy your contract using **GenLayer Studio**:

1. Open **[GenLayer Studio](https://studio.genlayer.com/)**.
2. Click **Create File** or import `contracts/truthstake.py`.
3. Verify that the compiler detects the correct header:
   ```python
   # v0.2.16
   # { "Depends": "py-genlayer:1jb45aa8ynh2a9c9xn3b7qqh8sm5q93hwfp7jqmwsfhh8jpz09h6" }
   ```
4. Select `Contract` in the compilation output panel.
5. Click **Deploy**.
6. Once deployed, copy the **Contract Address** generated by GenLayer Studio. You will need it for the frontend configuration.

## 2. How to Run the Frontend

### Prerequisites
Make sure you have **Node.js** (v18 or higher) and **npm** installed on your machine.

### Installation Steps

1. Navigate to the `frontend` folder:
   ```bash
   cd D:\Gen\TruthStake\frontend
   ```

2. Install the required dependencies:
   ```bash
   npm install
   ```

3. Configure your contract address. Open the `D:\Gen\TruthStake\frontend\.env` file and paste your deployed contract address:
   ```env
   VITE_CONTRACT_ADDRESS=0xYourDeployedContractAddressHere
   ```

4. Start the Vite local development server:
   ```bash
   npm run dev
   ```

5. The terminal will output the local port (usually `http://localhost:3000`). Open this URL in your web browser.

## 3. How to Use the dApp

1. **Connect Account**: 
   Click the **Connect Wallet** button. If you have the GenLayer wallet extension installed, it will connect. Otherwise, the frontend will automatically create a secure ephemeral local account for you and save it in your browser's local storage.
2. **Create a Market**:
   Enter a statement you want evaluated (e.g. "SpaceX launched Starship Flight 5 successfully").
   Choose your expected verdict, duration in minutes, and deposit some GEN tokens.
   Add 3-5 news URLs verifying the claim from whitelisted domains (e.g., Reuters, BBC, AP).
   Click **Deploy Fact Market**.
3. **Bet on Outcomes**:
   Find an active market, enter a stake amount, and click **Bet TRUE** or **Bet FALSE** to back either side of the statement.
4. **Trigger Resolution**:
   Once the countdown ends, the market will show a **Resolve Market** button. Clicking this triggers the GenLayer validator network to scrape the news sources and reach consensus.
5. **Claim Payouts / Refunds**:
   If you bet on the winning side, click **Claim Payout** to withdraw your proportional share of the pool. If the resolution failed due to dead URLs or parsing bugs, click **Claim Refund** to retrieve your stake.
